# Getting Started with Linux

## Installing Linux

On Local Windows Laptop (using bootable drive) possible ways 
- Dual boot 
- removing windows and installing linux as primary os
- Enabling WSL (Windows subsystem for linux), probably a partitioned part for using linux using a shell alongside with windows without dual-booting
- Using Virtual Box (a hypervisor, used to create vms on the laptop, inside the vm linux can be installed)
- On cloud providers  -Getting a vm and installing linux on it 
- Vagrant(new thing for me)

---

## Diff in Linux & Windows 
- Windows comes with a commercial license( we often see the license key) while linux is open source (anyone can freely use it) GPL (General public license)
- Windows often used for general purpose like day to day tasks - web browsing, entertainment and playing games while linux is majorly used for development, networking and for programming 
- Windows version based - windows xp, 10, 11 while linux flavours or distro based - ubuntu, fedora, centos, mint, arch, kali linux

## Kernel, Bootloader & Shell

The kernel is the core part of a os, and can be accessed using the Shell - a program available in the os, which can be used to provide shell commands that talks with the kernel and do the things. 
A Shell is a gateway/bridge between the user and the kernel.

Bootloader is a kernel level process which loads all the necessary files to start the operating system.
Eg - Grub is a bootloader in linux (most popular one) GRand Unified Bootloader
